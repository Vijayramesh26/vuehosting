1. Create Vue Project

vue create <ProjectName>

2. Build the Project

npm run build

3. Create Firebase Project

  1. Open Firebase Console
  2. Login to your Google account
  3. Click Add Project
  4. Name it, then click Continue ‚Üí Create Project

4. Navigate to Vue Project

Make sure you're in your project folder before continuing.

5. Install Firebase Tools

sudo npm install -g firebase-tools

6. Login to Firebase

firebase login

Authenticate in the browser window that opens.

7. Initialize Hosting

firebase init hosting
  • Select an existing or new project
  • Set public directory to dist
  • Configure SPA: No
  • Automatic build: No
  • Overwrite index.html: No

8. Deploy

firebase deploy --only hosting
